Immersywna gra o czołgach w estetyce retro.
## Wersja v1.1.alpha
Celem tej wersji:
- Refactoring kodu
- Dodanie łatwej konfiguralności sterowania
- Zaimplentowanie podstawowej fizyki czołgu (bezwładność, tarcie, opór)
